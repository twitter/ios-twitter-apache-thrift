Enumeration

PreencodedContainer

A container allowing pre-encoded data to be placed in the encoding buffer insted of a value

Declaration

enum PreencodedContainer<T> where T : Decodable, T : Encodable

Overview

For example the first struct would take .encodedData([0, 57]) and this would allow the second struct to be decoded with a value of 57

struct SomeStruct: ThriftCodable {
    let value: PreencodedContainer<Int16>
}

struct AnotherStruct: ThriftCodable {
    let value: Int16
}
