decode(_:from:)

Decodes a top-level value of the given type from the given Thrift representation.

Declaration

func decode<T>(_ type: T.Type, from data: Data) throws -> T where T : ThriftDecodable

Return Value

A value of the requested type.

Parameters

type

The type of the value to decode.

data

The data to decode from.

Discussion

Note

An ThriftDecoderError if any value throws an error during decoding.
